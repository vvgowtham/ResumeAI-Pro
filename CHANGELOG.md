# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-07-11

### Added
- Initial release of ResumeAI Pro
- Dashboard with resume statistics and ATS score overview
- My Resumes page with grid/list views and full CRUD
- 50+ professional resume templates (25 ATS + 25 Attractive)
- Live Resume Builder with 3-panel editor
- ATS Score Checker with detailed analysis
- Job Match Analyzer with keyword comparison
- Cover Letter Generator with multiple tones
- LinkedIn Profile Optimizer
- AI Studio with multi-provider support
- Resume Version Control
- Multi-format export (PDF, DOCX, PNG, JPG, HTML, MD, JSON)
- Analytics dashboard
- Billing page with tiered pricing
- Settings with profile and AI provider configuration
- GitHub Actions CI/CD pipeline
- Docker support
- Full monorepo structure
